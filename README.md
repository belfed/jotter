# Jotter

A minimal personal capture tool designed to reduce cognitive load. Jot down thoughts quickly, process them later.

## The problem

Ideas, tasks, and reminders pop up at the worst times. You need a place to dump them instantly — without friction — and organize them when you're ready.

## What it does

- **Inbox** — capture anything with a global command bar (`Ctrl+K`). Process items later by converting them into tasks (or future types like events, notes, etc.)
- **Tasks** — manage what needs to get done, with due dates and descriptions
- **Dashboard** — see what needs attention: overdue/upcoming tasks and unprocessed inbox items

## Stack

- **Next.js** (App Router, Server Actions)
- **React 19** + **TypeScript**
- **Prisma** + **PostgreSQL**
- **Tailwind CSS** + **shadcn/ui**
- **next-intl** (i18n)
- **next-themes**
